# Fase de Diseño (RUP)

Esta carpeta contiene las decisiones de **solución técnica** y la arquitectura detallada del sistema.

## Objetivos
- Aterrizar los requisitos del análisis a una implementación concreta.
- Definir la estructura técnica y los patrones de diseño aplicados.
- Servir como guía directa para la fase de implementación (Codificación).

## Contenido
- **Arquitectura de Capas**: Organización lógica del software.
- **Diagramas de Clases de Diseño**: Detalle de atributos, métodos y tipos de datos.
- **Diagramas de Secuencia Detallados**: Interacciones técnicas cronológicas que incluyen manejo de excepciones y lógica de infraestructura.
